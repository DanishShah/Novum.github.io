How Novum works?

Novum works differently on different UI's based on how it is configured.
Consider an example of the UI below:


Original

Adapted

The UI represents a simple application which has a set of actions that the user can perform.
There are no changes since the UI is not adapting. As the user uses the application, Novum store the actions performed by user and keep a track of that and starts learning.
Once it has collected enough data, it starts adapting.
In the example below, it is assumed that the user uses ACTION 3 more frequently than the other actions. Novum adapts accordingly and replaces the action where it can be accesseble more easily by the user constructing a new personalized UI.
